## kneunginfo-new
이 강중인포는 강릉중학교 학생들의 학교생활을 더욱 편하게 할 수 있도록 자가진단 바로가기, 시간표 보기, 급식 보기, 학사일정 등 여러 편의기능을 한번에 모아둔 웹페이지입니다.

## 사용방법
1. PHP, DB가 설치되어있는지 확인합니다.
2. db.php 파일과 /member/page/main/db.php 파일을 열어 DB 정보를 입력합니다.
3. 파일을 서버에 업로드합니다.
